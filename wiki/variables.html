Creating and storing variables

Variables

In simple line, you can use the store function to save and set variables. 

10 > store x

You can also easily add to a variable without having to spell it twice. 

10 > store reallyLongVariableName
reallyLongVariableName > + 10 > store

There is a very special case that allows you to get and set variables using strings 

|name| > store x
10 > store [x]
