Simple Line Documentation

Introduction

Simple Line is a minimalist programming language designed for clarity, readability, and linear structure. Its syntax emphasizes clean logic and minimal punctuation.

Installation

To use Simple Line, make sure you have node, go to the command terminal, and run: 

cd (the folder location where you want to run your code)

this will take you to the folder where you want to install SimpleLine:

npm init -y
npm install simpleline@latest

next, make a .smplin file, open it, and put the code in. Then, run it with:

simpleline (file name).smplin

you can also run code with javascript using:

const sl = require("simpleline"); //requires simpleline to be downloaded in the file with npm
sl.run(`
code
`);
